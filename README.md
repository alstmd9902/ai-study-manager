# 📘 수업 관리 웹앱 (AI Study Manager)

1인 사용 전용 **수업 기록·학생 숙제(단어 점수)·주간 평균 관리** 웹앱입니다.  
주차 단위로 수업 데이터를 관리하며, 새로고침 후에도 기록이 유지됩니다.

---

## ✨ 주요 기능

### 📅 주차 관리
- 날짜(Date 기준)로 주차 생성 및 선택
- 주차별 데이터 자동 로드
- 선택 주차 삭제 (하위 기록 포함)

### 📆 요일 / 교시 관리
- 요일 그룹: **월·수·금 / 화·목·토**
- 교시 선택: **1교시 / 2교시 / 3교시**
- 요일·교시별 독립 데이터 관리

### 📝 교시별 수업 입력
- **특이사항**: 결석, 보충일자, 기타 메모
- **진도 관리**: 리딩 / 리스닝 / 문법
- **숙제 관리**
  - 학생별 단어 점수 입력
  - 100% 미만 사유 기록
  - 이번 주 이슈 및 못한 숙제 관리

### 👩‍🎓 학생 주간 요약
- 주차별 학생 숙제 평균 자동 계산
- **평균 계산 기준**
  - 해당 주차의 모든 교시 중
  - 점수가 입력된 교시만 포함
  - 소수점 1자리 반올림
- 100% 미만 사유 / 주간 이슈 요약 표시

### 📤 데이터 출력
- **Excel 다운로드**
  - 수업 기록 시트
  - 학생 주간 요약 시트
- **PDF 다운로드**

### 💾 데이터 저장
- LocalStorage 사용 (`weekly-records`)
- 새로고침 및 재접속 시 데이터 유지

---

- **Frontend**: React 19, Vite, TypeScript
- **UI**: Tailwind CSS v4, Lucide React
- **파일 출력**
  - Excel: xlsx
- **배포**: Vercel
